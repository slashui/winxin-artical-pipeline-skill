# 封面图 Prompt 模板

## 用法

生成封面图时，按下面结构拼接：

1. 内容段
2. 固定风格段
3. 负面约束

最终格式：

```text
[内容段]

[固定风格段]

[负面约束]
```

---

## 内容段模板

```text
画面主体是[人物/对象]，主视觉是[谁]，副视觉是[谁]。画面围绕[主题]展开，背景中融入[符号元素]，整体传达[情绪/冲突]。
```

### 内容段示例

```text
画面主体是两位访谈人物的大幅半身头像拼贴，主视觉是 Simon Willison，副视觉是播客主持人。画面围绕 AI 编程时代转折展开，背景中融入终端窗口、测试通过标记、agent workflow、流程箭头、代码块，整体传达软件工程正在被重写的张力与思想交锋。
```

---

## 固定风格段

```text
横版 16:9，丝网印刷宣传画风格的 editorial illustration，适合微信公众号封面。像一张关于时代转折的严肃人物访谈海报。整体采用强烈套色与有限色盘：高饱和红色、橙色、米白色、深蓝黑色，颜色硬切分区，带明显丝网印刷颗粒、油墨质感、轻微错位套印、纸张纹理。背景中融入抽象的主题符号元素，被处理成海报图形语言，而不是真实 UI 截图。整体具有强对比、号召感、历史感和思想交锋感。人物要有真实肖像特征，但保留插画化处理，不要摄影写实，不要卡通，不要赛博朋克，不要生成任何文字、logo、水印，为中文标题预留清晰留白区域。
```

---

## 负面约束

```text
不要机器人，不要机械手，不要赛博朋克城市，不要紫色霓虹，不要通用科技背景，不要乱码文字，不要英文海报标语，不要夸张笑脸，不要卡通，不要低清晰度，不要多余人物，不要手部畸形，不要脸部变形，不要真实照片感，不要现代发布会背景，不要玻璃拟态 UI。
```
